## Commeit Message

### 結構
```
<­typ­e>­[o­ptional scope]: <de­scr­ipt­ion­>

[optional body]

[optional footer]
```

## 類型

|          |                         |                    |
|----------|:-----------------------:|-------------------:|
| feat     |Features                 | 新功能 |
| fix      |Bug Fixes                | 錯誤修正 |
| docs     |Documentation            | 文件 |
| style    |Styles                   | 樣式 |
| refactor |Code Refactoring         | 重購 |
| perf     |Performance Improvements | 效能相關 |
| test     |Tests                    | 新增或修正測試 |
| build    |Builds                   | 建置相關工具 |
| ci       |Continuous Integrations  | 持續整合相關 |
| chroe    |Chroes                   | 其他修正不關乎程式碼 |
| revert   |Reverts                  | 回復版本 |
